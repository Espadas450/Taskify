#Taskify - Task Management System

Taskify is a Java application with Spring Boot created as a programming test for a company.

## Requirements

- *Java 23*
- *Maven*
- *Spring Boot 3.4.0*

## Dependencies

- *Spring Boot DevTools*
- *Spring Web*
- *Spring Data JPA*
- *PostgreSQL Driver*
- *Validation*
- *Lombok*
